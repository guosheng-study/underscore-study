3.7 - without(排除)

语法: _.without(array, *values)

关键词：排除。

返回一个除去所有values后的数组

a) - _.without()示例
var arr1 = [1, false, NaN, 0, 3, 1, 4]; //过滤数组里0和1的值。 var res1 = _.without(arr1, 0, 1); console.log(res1); //参数的个数是不定的。 var res4 = _.without(arr1, 0, 1, false); console.log(res4);
