3.6 - flatten(去除嵌套)

语法: _.flatten(array, [shallow])

关键词：变平、去除嵌套。

将一个嵌套多层的数组 array (嵌套可以是任何层数)转换为只有一层的数组. 如果传参 shallow 为true, 数组只转换第一层.

a) - _.flatten()示例
var arr = ['123', [ [1, 2], [2, 3], [3, 4] ], ['a', 'b'] ]; //变成一维数组 var res1 = _.flatten(arr); console.log(res1); //只去掉了一层 res2 = _.flatten(arr, true); console.log(res2);
