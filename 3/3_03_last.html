3.3 - last

语法: _.last(array, [n])

关键词：末尾取值。

返回一个数组的最后一个元素。如果传了n,返回最后n个元素组成的数组。

a) - _.last()示例
var list = [1, 2, 3, 4]; //返回最后一个元素 var res1 = _.last(list); console.log(res1); //=> 4 //使用原生方法写 var res2= list.slice(-1)[0]; console.log(res2); //=> 4
http://jsfiddle.net/kyo4311/kySg4/

不传n的时候，返回的是数组最后一个元素，不是一个数组。

b) - 传入n
var list = [1, 2, 3, 4]; //返回前面的元素，排除最后2个元素 var res1 = _.last(list, 2); console.log(res1); //=> [3,4] //当n大于数组长度时，整个数组 var res2 = _.last(list, 10); console.log(res2); //=> [1,2,3,4] //传了n之后，返回一个元素组成的数组。 var res3 = _.last(list, 1); console.log(res3); //=> [4] //不传n，返回的是一个元素，不是数组。 var res4 = _.last(list); console.log(res4); //=> 4
http://jsfiddle.net/kyo4311/BPGm7/

当传入了n,返回后面n个元素组成的数组，n传1，返回的也是数组。
