3.1 - first

语法: _.first(array, [n])

关键词：开头取值。

返回列表中的值的数目。

a) - _.first()示例
var list = [1, 2, 3, 4]; //返回一个的时候，不是数组。 var res1 = _.first(list); console.log(res1); //=> 1 //用原生js写 var res2 = list.slice(0, 1)[0]; console.log(res2); //=> 1 //返回多个的时候是数组。 var res3 = _.first(list, 2); console.log(res3); //=> [1, 2] //用原生js写 var res4 = list.slice(0, 2); console.log(res4); //=> [1, 2]
http://jsfiddle.net/kyo4311/a3bYY/

_.first()传一个list，则返回list的第一个元素，返回的结果不是数组，如果第二个参数n也传了，则返回一个数组。
