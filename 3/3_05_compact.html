3.5 - compact

语法: _.compact(array)

关键词：保留真值。

保留数组里的真值（可以理解为==true），返回一个新数组。 在javascript里假值有， false, null, 0, "", undefined和NaN。

a) - _.compact()示例
var arr = [0, 1, false, 2, '', 3, undefined, NaN]; //保留真值 var res1 = _.compact(arr); console.log(res1); //=> [1, 2, 3] //用filter来实现他. var res2 = _.filter(arr, function (n) { return n && true; }); console.log(res2); //=> [1, 2, 3]
