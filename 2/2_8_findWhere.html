2.8 - findWhere(特性查找第一项)

语法: _.findWhere(list, properties)

关键词：按特性查找第一项。

筛选集合中符合指定条件的第一个元素并返回新的结果。list为筛选的目标集合；properties键值对(指定条件)，如{sex:'Male'},若为多个，则以逗号分隔， 如{sex:'Male',age,10}

a) - 从list中筛选出有properties特性的第一个元素
var list = [{ 'name': 'guofucheng', 'sex': 'Male' }, { 'name': 'fanbingbing', 'sex': 'Female' }, { 'name': 'liudehua', 'sex': 'Male' }]; var properties = {'sex' : 'Male'}; //筛选出list集合中sex为Male的第一个元素。 var stars = _.findWhere(list, properties); console.log(stars); //=> Object {name="guofucheng", sex="Male"}

_.where()方法返回的是一个数组，而_.findWhere()和_.find()一样，返回都只是list里的一个元素，都不是数组。下面我们用_.find()来实现同样的功能。

b) - 用find实现同样的功能
var list = [{ name: 'guofucheng', sex: 'Male' }, { name: 'fanbingbing', sex: 'Female' }, { name: 'liudehua', sex: 'Male' }]; //筛选出list集合中sex为Male的第一个元素。 var star = _.find(list, function (v) { return v.sex === 'Male'; }); console.log(star); //=> Object {name="guofucheng", sex="Male"}
