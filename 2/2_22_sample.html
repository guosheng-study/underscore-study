2.22 - sample

语法: _.sample(list, [n])

关键词：取样。

从列表里随便抽样，返回新的结果，n是返回的抽样个数。n可以省略，则返回一个元素。

a) - sample示例
//取样，从列表中随机取出一个值 var sample1 = _.sample([1, 2, 3, 4, 5, 6]); console.log(sample1); //=> 2 //取样，从列表中随机取出3个值 var sample2 = _.sample([1, 2, 3, 4, 5, 6], 3); console.log(sample2); //=> [1, 6, 4]
http://jsfiddle.net/kyo4311/fbxN5/

返回的结果是每次都是随机的，当n不传的时候，返回的是一个结果，如果n传到大于1的时候，返回的是一个数组。

b) - list可以是字符串或对象
var listStr = '1234567890'; var listObj = { a: 1, b: 2, c: 3, d: 4, e: 5 }; console.log(_.sample(listStr, 3)); //["1", "4", "5"] console.log(_.sample(listStr, 3)); //["8", "6", "1"]
http://jsfiddle.net/kyo4311/5LL8V/

list可是数组、对象和字符串。
