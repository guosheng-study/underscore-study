2.9 - reject(排除)

语法: _.reject(list, predicate, [context])

关键词：排除、去除。

筛选并返回集合中不符合条件的元素，与_.filter()相反。

a) - 数组排除
var evens = _.reject([1, 2, 3, 4, 5, 6], function (num) { return num % 3 === 0; //排除能被3整除的元素 }); console.log(evens); //=>[1, 2, 4, 5]
http://jsfiddle.net/kyo4311/24Jx3/

b) - 对象排除
var evens = _.reject({ a: 1, b: 2, c: 3, d: 4 }, function (num) { return num % 2 === 0; //排除能被2整除的元素 }); console.log(evens); //=>[1, 3]
http://jsfiddle.net/kyo4311/hbLH3/

注意，在操作对象的时候，还是返回了对象的值，并不像{a:1, c:3}这样子。context参数可以改变predicate里面this的指向。predicate返回的参数和_.filter()一样，第一个参数为list里的元素，第二个参数为元素对应list的index，第三个参数为list本身。如有不明之处可以查看_.filter()
