2.14 - pluck(采集)

语法: _.pluck(list, propertyName)

关键词：采集、提取。

从list里面提取指定属性的值。

a) - 提取属性的值
//收集属性为name的值。 var stooges = [{ name: 'moe', age: 40 }, { name: 'larry', age: 50 }, { name: 'curly', age: 60 }]; var pluck = _.pluck(stooges, 'name'); console.log(pluck); //=>["moe", "larry", "curly"]
http://jsfiddle.net/kyo4311/a7PKc/

_.pluck()比较简单，第一个参数为list,传入数组，数组里的元素是对象，第二个参数只要传入需要提取的值的属性名(键名)

b) - 用_.map()方法实现他
//收集属性为name的值。 var stooges = [{ name: 'moe', age: 40 }, { name: 'larry', age: 50 }, { name: 'curly', age: 60 }]; var pluck = _.map(stooges, function(v){ return v.name; }); console.log(pluck); //=>["moe", "larry", "curly"]
http://jsfiddle.net/kyo4311/wGpV2/

_.pluck()能实现的时候就不必再用_.map()，灵活应用这些方法，可以让代码看上更加优雅。
