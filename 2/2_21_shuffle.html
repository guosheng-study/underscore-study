2.21 - shuffle

语法: _.shuffle(list)

关键词：洗牌、打乱次序。

将list里的元素打乱次序，然后返回。使用Fisher–Yates shuffle算法。

a) - shuffle示例
//洗牌，随机打乱顺序 var list = [1, 2, 3, 4, 5, 6]; var res = _.shuffle(list); console.log(list); //[1, 2, 3, 4, 5, 6] 原来的list不变 console.log(res); //[4, 1, 3, 5, 2, 6] 每次结果是不一样的。
http://jsfiddle.net/kyo4311/b4gQc/

_.shuffle()方法传入list，返回洗牌后的新数组。原来的list不变，新的数组每次结果都可能不一样。

b) - list可以是字符串或对象
var listStr = '123456'; var resStr = _.shuffle(listStr); console.log(resStr); //["2", "1", "6", "5", "3", "4"] //返回了一个数组 var listObj = { a: 1, b: 2, c: 3, d: 4, e: 5 }; var resObj = _.shuffle(listObj); console.log(resObj); //[1, 5, 3, 2, 4] 返回了一个数组，值为对象里的值
http://jsfiddle.net/kyo4311/NGUP5/

list可是数组，还可以是对象，但是他们返回的都是数组，当list是对象的时候，返回的数组里的元素是list元素的值。
