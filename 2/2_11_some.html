2.11 - some(一些)

语法: _.every(list, [predicate], [context])

名别: any(任何一个)

关键词：任何一个。

如果list中的元素只要有一个符合predicate条件，则返回true，都不符合才返回false。一旦找到了符合条件的元素, 就直接中断

a) - some示例
//测试是不是有一个值大于4,返回值是布尔值 var res = _.some([1, 2, 3, 4, 5, 6], function (n) { console.log(n); //=> 1 //=> 2 //=> 3 //=> 4 //=> 5 return n > 4; }); console.log(res) //=> true
http://jsfiddle.net/kyo4311/Jyf2M/

只要有一项大于4则返回true，我们发现predicate里面的6并没有出现，因为一旦找到了符合条件的元素, 就直接中断，然后回结果了

学习_.some()的和结合_.every()一起学习，predicate是可以省略的，context则可以改成predicate里面的this指向。而predicate和_.every()一样，有三个返回的参数。
