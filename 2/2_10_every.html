2.10 - every(每项)

语法: _.every(list, [predicate], [context])

名别: all(全部)

关键词：每项、全部。

如果list中的所有元素都符合predicate条件，则返回true，否则返回false

a) - 数组的每项都符合条件
var res = _.every([1, 2, 3, 4, 5, 6], function (num) { return num > 0; }); console.log(res); //=> true
http://jsfiddle.net/kyo4311/Jyf2M/

上例数组中每项元素都大于0，因此返回true。

b) - 当有一个条件不符合即中断
var res = _.every([1, -2, 3, 4, 5, 6], function (num) { console.log(num); //=> 1 //=> -2 return num > 0; }); console.log(res); //=> false
http://jsfiddle.net/kyo4311/K5mks/

上例中,-2已经不大于0了，所以就中断了，直接返回了false，如果我们反过来理解，只要有一项不符合条件就返回false,只有全符合了才返回了true。

c) - predicate参数也是可以省略的
var res1 = _.every([1, -2, 3, 4, 5, 6]); console.log(res1); //=> true var res2 = _.every([1, false, 3, 4, 5, 6]); console.log(res2); //=> false
http://jsfiddle.net/kyo4311/bqSU9/

predicate没有传值的时候，直接判断list里的值每一项是不是为true如果每一项都为ture则返回true,如果有一项为false则返回false。例子为false，当然可以是null、NaN、undefined、0等。

d) - context参数说明
var context = { a: NaN, b: 0, c: false }; var res = _.every(['a', 'b', 'c'], function (n) { console.log(n, this); //=> a Object {a=NaN, b=0, c=false} //=> b Object {a=NaN, b=0, c=false} //=> c Object {a=NaN, b=0, c=false} return this[n] || false === false; }, context); console.log(res) //=>true
http://jsfiddle.net/kyo4311/PBLSq/

当传入context之后，predicate里面的this即是context。context里面的值每一项都==false所以返回了true。

e) - predicate参数说明
var res = _.every([2, 4, 6, 8], function (n, index, list) { console.log(n, index, list); //=> 2 0 [2, 4, 6, 8] //=> 4 1 [2, 4, 6, 8] //=> 6 2 [2, 4, 6, 8] //=> 8 3 [2, 4, 6, 8] return list[index] % 2 === 0; }); console.log(res) //=>true
http://jsfiddle.net/kyo4311/MqvhL/

predicate返回了三个参数，第一个参数为元素里的值，第二个参数为元素对应list的序号index,第三个参数为list本身。
