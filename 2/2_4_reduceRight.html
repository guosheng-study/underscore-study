2.4 - reduceRight(右精简)

语法: _.reduceRight(list, iterator, memo, [context])

别名: inject, foldl

关键词：右合并、精简、归一。

a) - 用reduceRight精简数组
var list = [ [0, 1], [2, 3], [4, 5] ]; var flat = _.reduceRight(list, function (a, b) { return a.concat(b); //返回数组 }); console.log(flat); //= > [4, 5, 2, 3, 0, 1]
http://jsfiddle.net/kyo4311/wsWu2/

_.reduceRight()方法和_.reduce()是类似的，唯一的区别他是从右边开操作的，看取名也是很好理解，而上例中，我们返回值的时候用了a.concat(b)，所以最后的值才会是一个数组，而_.reduce()例子里中都只是一个简单的值。

b) - list其实也可以一个字符串
var flat = _.reduceRight('0123456789', function (a, b) { return a.concat(b); //返回数组 }, []); //试试这里的数组不传会如何 console.log(flat); //= > ["9", "8", "7", "6", "5", "4", "3", "2", "1", "0"]
http://jsfiddle.net/kyo4311/fB95s/

我们知道list的地方需要传入集合，事实我们比较清楚的合集有两种，1、数组[1,2,3]；2、对象{a:1,b:2,c:3}。而在上例中list传入了一个字符串。另外在这里也介绍另外一种集合：arguments。那看下例。

c) - list还可以是arguments
var arr = [1, 2, 3, 4, 5]; var func = function () { console.log(arguments); //=> [1, 2, 3, 4, 5] var flat = _.reduceRight(arguments, function (a, b) { return a + b; }); console.log(flat); //=> 15 }; func(1, 2, 3, 4, 5);
http://jsfiddle.net/kyo4311/jh2hx/

JavaScript 中每个函数内都能访问一个特别变量 arguments。这个变量维护着所有传递到这个函数中的参数列表。如果需要了解更多请查看http://bonsaiden.github.io/JavaScript-Garden/zh/#function.arguments 而本例作者只想说明集合也包括了arguments。
